Data Visualization Project

116th Senate Twitter Follower Sharing Ego-Network


Currently Under Construction


As of April 2020, about 23 million unique twitter users follow members of the 116th U.S. Senate. This project aims to interatively visualize ego-networks that allow you to explore which Senators share supporter base (at least on Twitter).


R codes used to generate this interactive plot will be available soon through my github account.
