116th Senate Twitter Follower Sharing Ego-Network


[R Shiny Interactive Plot, Version 0.5]

As of April 2020, about 23 million unique twitter users follow members of the 116th U.S. Senate. This interactive visualization project allows you to explore which Senators share twitter followers. In this project I created a R Shiny App to interatively visualize ego-networks which allow you to explore which Senators share supporter base (at least on Twitter).


R codes used to generate this interactive plot will be available soon through my github account.
